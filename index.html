PROJECTS

ENDEAVR One, Spring 2020

A machine learning project, in conjunction with the CSCE 421 course and Texas A&M's ENDEAVR Project, that allows the city of Nolanville, TX to gather information regarding the city by utilizing SMS text messages without the need for an internet connection. It utilizes Twilio Autpilot and a Flask-based back-end to process text messages with machine learning, a MongoDB Atlas database, and the Google Cloud Platform API for directions and the AccuWeather API for forecasting and weather data. This project, among others in the organization of ENDEAVR One, allows the city to push towards a "smart city" platform, where there is constant collaboration between students majoring in Computer Science, Landscape Architecture, Urban Planning, and Visualization.

Relevant skills: Twilio Autopilot, Python, Flask, MongoDB, RESTful API
